# hudukaata

## Session Start

When starting a session to develop a feature, read `.kdevkit/feature-dev.md` and follow the workflow defined there.

## Mandatory Dev Loop

Whenever you implement any code change, you MUST run the full agent development loop defined in `.kdevkit/agent-dev-instructions.md`. This is not optional — every code change, no matter how small, requires completing the Setup → Code → Quality → Code → Test → Push loop before the task is considered done.

## References

See `.kdevkit/agent-dev-instructions.md` for the mandatory development loop.

See `.kdevkit/project.md` for project context, quality threshold, and max test retry settings.
